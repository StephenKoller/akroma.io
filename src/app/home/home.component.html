What is Akroma?

Akroma is a cryptocurrency and application development platform focused on creating a governance model that will reward long-term, sustainable development and design.

Masternodes

Masternodes help protect and scale the Akroma network by validating transactions and acting as a decentralization incentive. Running a masternode is rewarded with Akroma.

The more nodes a cryptocurrency network has, the more prone it becomes to tampering and attacks.

To protect against 51% attacks coming from centralized miner farms, the incentives for running a node and mining have been unbundled.

Decentralized Oracles

Next generation smart contracts powered by real-world data feeds.

Smart contracts aren't so smart without data from the real world. When you execute a smart contract, you're passing tokens around which have real value. Oracles therefore need to be trusted, and so we'll be using practical Byzantine fault-tolerance (PBFT).
